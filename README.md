# national_prayer_breakfast_speech_data
Transcripts from every speech given by a sitting president at the National Prayer Breakfast (1971-2026), as well as a paragraph-by-paragraph topic classification of Trump's 2026 speech.

If you use this data, I would love to know how you analyzed it! DM me on TikTok or Instagram @crosstabmedia

Data manually collected from The American Presidency Project at UC Santa Barbara: https://www.presidency.ucsb.edu/documents/app-attributes/national-prayer-breakfast
